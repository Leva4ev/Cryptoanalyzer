Техническое задание
Разработать программу, позволяющую шифровать и расшифровывать текст с использованием шифра Цезаря.

Программа должна поддерживать несколько режимов работы:
 1) Расшифровка текста с помощью ключа
 2) Шифровка текста
 3) Расшифровка текста с помощью brute force (перебор всех вариантов) - сделать диапозон, а лучше логику с переворотом колеса
 - программа должна открывать указанный пользователем файл с текстом
 - проделывать с ним одно из указанных выше действий.
 - после этого создавать новый файл с результатом.
 - обеспечивать обработку больших файлов
 - иметь валидацию входных данных.

Опционально:
 - добавить графический интерфейс пользователя, а также
 - добавить статистический анализ для автоматического взлома шифра.
